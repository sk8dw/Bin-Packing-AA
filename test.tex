% ---------- Section ----------
\section{Introduction}

\LaTeX{} is a document preparation system designed for producing
high-quality technical and scientific documents.
Unlike word processors, it rewards precision and punishes improvisation.

This document is organized as follows:
Section~\ref{sec:math} covers mathematics,
Section~\ref{sec:tables} introduces tables,
and Section~\ref{sec:figures} deals with figures.

% ---------- Math ----------
\section{Mathematics}
\label{sec:math}

Inline math looks like this: \( a^2 + b^2 = c^2 \).

Displayed equations look like this:

\begin{equation}
E = mc^2
\label{eq:einstein}
\end{equation}

Equation~\ref{eq:einstein} is suspiciously popular.

You can also align equations:

\begin{align}
f(x) &= x^2 + 2x + 1 \\
     &= (x + 1)^2
\end{align}

Tables should never look like spreadsheets.

\begin{tabular}{lcc}

Item & Value A & Value B \\
\midrule
Alpha & 10 & 20 \\
Beta  & 15 & 30 \\
Gamma & 42 & 84 \\
\bottomrule

% ---------- Figures ----------
\section{Figures}
\label{sec:figures}

Figures require image files.
If the file does not exist, LaTeX will complain loudly.

\begin{figure}[H]
\centering
\includegraphics[width=0.6\textwidth]{example-image}
\caption{Placeholder figure provided by LaTeX}
\end{figure}

If that image fails, install the package `mwe`, or comment this section out.
